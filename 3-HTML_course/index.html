Os primeiros computadores

A primeira geração:

Os primeiros computadores criados foram feitos por volta do fim da primeira metáde do séc. XX, sendo um deles o grande Harvard Mark I que era tido, na verdade, como uma calculadora automatizada. Após ele, tivemos também o Colossus e o ENIAC, este último sendo o primeiro a ser programado de fato, embora de uma maneira ainda muito peculiar à que existe hoje em dia. Esses computadores eram fiéis ao seu nome, no sentido de que eram apenas gigantescas máquinas de cálculo, usados primariamente como calculadoras ou decifradores de enigmas, mensagens codificadas e por isso, são chamados de 1ª geração dos computadores. Fora durante essa geração de computadores que o termo bug foi incorporado ao mundo da computação.

A segunda geração:

A segunda geração de computadores fora marcada pela drástica diminuição do tamanho das máquinas, diminuição que fora proporcionada pela tecnologia dos transistores, que tornara possível a sua locomoção, algo que era impossível devido ao tamanho dos computadores, como ilustra a imagem acima; no entando, ainda não tinham uso tão comercial quanto aqueles presentes na terceira geração.

A terceira geração:

A terceira geração se destaca por ter diminuido ainda mais o tamanho dos equipamentos, dessa vez por conta da tecnologia dos circuitos integrados. Tornando-os aind amais portáteis e aumentando sua capacidade computacional.

A quarta geração:

Essa é a geração da qual fazemos parte. A portabilidade foi elevada a níveis nunca antes vistos, sendo nós hojes capazes de portar um computador dezenas de vezes mais poderoso que os grandes da primeira geração em nossos bolsos, através dos smartphones sendo os pioneiros da inovação empresas como a Microsoft de Bill Gates e a Apple de Steve Jobs.



Como surgiu

As primeiras redes

Em meados do sec. XX, o fim da segunda Guerra Mundial levou à Guerra Fria entre os Estados Unidos da América ea União Soviética, o levou assim à criação da DARPA que trabalhou no projeto de comunicações chamado Arpanet, que foi a primeira rede de comunicação de computadores a nível nacional, projeto que teve a participação de Joseph Carl LickLider aquele seria responsável por dar a primeira ideia de uma núvem. A partir do aumento da rede da Arpanet e do surgimento de várias outras rede, também para fins militares, Vinton Cerf e Robert Kahn trabalharam e desenvolveram os protocolos de comunicação TCP/IP. Fora apenas em 1989 que, com o World Wide Web criado por Tim Berners-Lee que as redes de computadores foram popularizadas fora do âmbito militar e foi adotada por todo o Estados Unidos.

Client no Linguajar Web

Client é uma palavra utilizada sempre em contraste com servidor, pois denota uma relação, em que os clients são máquinas utilizadas para acessar informações contidas nos servidores, em se tratando de aplicações Web, a operação é feita pelos Navegadores. Quando se trabalha com clients, a maneira de tra balho é diferenciado do que seria se você estivesse trabalhando com servidores.

Navegadores

Navegadores são os programas utilizados para interpretar os arquivos Web solicitados(HTML, CSS e JavaScript). No entanto, hoje em dia, os browser estão sendo ampliados para interprertarem uma gama maior de arquivos, e. g., JSON, PDF. Existe um consórcio mundial para a padronização dos navegadores que é liderado por Tim Berners-Lee, o W3C.

Aplicações Web

Aplicaçõe Web são, praticamente, todos os sites da atualidade. Pois são aplicações, mas que não é necessária a sua instalação no copmputador, utilizando a internet como meio de comunicação.

O que são servidores

Servidores são computadores especializados no armazenamento e proveniência de dados. Esses computadores podem estar ligados à internet, como é o caso de aplicações web que necessitam de acesso a um banco de dados, ou não, como é o caso de servidores locais em empresas e locais de trabalho. Eles se utilizam de sistemas operacionais próprios, e. g., Windows Server, Red Hat, CentOS, etc. Podem existir 5 tipos de servidores: de Arquivos, de Segurança ou Firewall, Streamin, e-mail e Web. Um grande conglomerado de servidores é chamado

Hardware e Softwares para servidores

Toda máquina é composta por duas partes: a lógica e a física. Em ambas as partes o servidores se diferenciam dos desktops normais pois além de, como dito acima, eles possuírem sisrtemas operacionais próprios, também possuem um hardware diferenciado, mais poderoso, para que o processamento de dados seja mais rápido e não se perca performace

Tipos de Servidores

  • Proxy
    • Bloqueia o acesso dos clients a certos servidores
  • Firewall
    • Bloquei o acesso de certas máquinas externas às internas
  • Web
    • Servidores que hospedam aplicações web
    • Utilizam geralmente programas para que isso aconteça, como Apache Server
  • E-mail
    • Usado para armazenar correio eletrônico
  • Database
    • Armazenam dados para aplicações web
  • DNS
    • É o servidor responsável por armazenar e integrar os nomes dos sites com os seus ips

O que é?

"Linguagem de programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas(softwares)"

Server-Side

São linguagens de programação que rodam do lado do servidor, isto é, que trabalhan especificamente com o servidor. Alguns exemplos são o PHP, C# e Ruby.

Client-Side

São linguagens de programação que rodam do lado do client. A mior delas é o JavaScript.

HTML não é linguagem de programação

HTML não é uma linguagem de programação, é apenas uma linguagem de marcação para a interpretação de textos por parte do navegador.
